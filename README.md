# AI Resume & Portfolio Builder

An AI-powered web application that generates professional, ATS-optimized resumes using OpenAI and Streamlit.

## Features
- Generates resume in seconds
- ATS-optimized formatting
- DOCX download option
- Customizable job role targeting

## Tech Stack
- Python
- Streamlit
- OpenAI API
- python-docx

## How to Run Locally

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Set your OpenAI API key:
   export OPENAI_API_KEY=your_api_key
4. Run:
   streamlit run app.py
